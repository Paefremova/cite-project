Today I completed the control activities for the external Stepik course "Basics of Cybersecurity". The work focused on three main thematic blocks: network security, device protection, and practical cryptography-related concepts.

From a methodological point of view, this course was useful because it combined short theoretical explanations with test questions and interactive tasks. Such a format made it possible not only to memorize individual facts, but also to trace the logic behind them: how network protocols work, why secure connections matter, how passwords should be stored, and which threats are most common in everyday digital environments.

### Network Security Foundations

The first block examined the foundations of secure network communication. In particular, I reviewed the distinction between common protocols, the role of HTTPS and TLS, the function of cookies, and the basic principles of anonymity tools and Wi-Fi security. These topics are important because they describe the boundary between ordinary internet use and informed security-aware behavior.

### Device Protection

The second block was devoted to protecting a computer or phone. It covered disk encryption, password hygiene, phishing, malware examples, and secure messaging. This part was especially practical, since it connected theoretical security concepts with routine user behavior. In digital environments, technical protection is not enough by itself; user discipline remains a critical factor.

### Reporting and Documentation

I also prepared the reporting materials and presentation structure for the course. This stage required careful selection of screenshots, concise explanations for each completed activity, and a clear separation between report-style documentation and presentation-style summarization. That distinction turned out to be important: the report must prove completion, while the presentation must communicate the key ideas briefly and coherently.

{{% callout note %}}
**Key takeaway:** Cybersecurity is not a single tool or a single habit, but a system of small correct decisions. Strong passwords, secure connections, cautious verification of links, and attention to device protection all contribute to a more reliable digital environment.
{{% /callout %}}

Today's work was not only a technical exercise, but also a useful reminder that digital security begins with everyday reasoning.
